# Falling Blocks Puzzle

HTML / CSS / JavaScriptだけで作った、ブラウザで遊べる落ちものブロックパズルです。外部ライブラリは使っていないため、GitHub Pagesへそのまま公開できます。

## 起動方法

1. このフォルダの `index.html` をブラウザで開きます。
2. ローカルサーバーを使う場合は、このフォルダで任意の静的サーバーを起動して `index.html` にアクセスします。

例:

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開きます。

## 操作方法

| 操作 | キーボード |
| --- | --- |
| 左右移動 | ← / → |
| ソフトドロップ | ↓ |
| 回転 | ↑ / X |
| ハードドロップ | Space |
| ホールド | C |
| 一時停止 | P |
| リスタート | R |

スマホ・タブレットでは画面下の操作ボタンで遊べます。

## GitHub Pagesで公開する

1. このフォルダをGitHubリポジトリへpushします。
2. GitHubのリポジトリ設定で `Pages` を開きます。
3. `Deploy from a branch` を選び、公開したいブランチのルートを指定します。
4. 発行されたURLへアクセスします。

## ファイル構成

- `index.html`: 画面構造
- `style.css`: レスポンシブUIとタッチ操作向けスタイル
- `script.js`: ゲームロジック、描画、入力処理
