# Falling Blocks Puzzle

HTML / CSS / JavaScriptだけで作った、ブラウザで遊べる落ちものブロックパズルです。外部ライブラリは使っていないため、GitHub Pagesへそのまま公開できます。

## モード

- ノーマルモード: 通常ルールで遊ぶ標準モードです。
- キッズモード: 落下速度がゆっくりで、文字やボタンを大きくした子ども向けモードです。ラインを消すと「ナイスプレー！」などの応援メッセージが出ます。
- 3分チャレンジ: キッズ向けの180秒チャレンジです。時間が終わると「しあい終了！」として、今回のスコア、消したライン数、ランク、ベストスコアを表示します。

キッズ系モードでは、野球風の言葉づかい、かわいい丸みのあるブロック、簡単な効果音、モード別ベストスコア保存に対応しています。

## ライン消去演出

ラインがそろった時は、すぐに消えずに一瞬光ってから縮み、フェードアウトして上のブロックが落ちます。キッズモードではブロックが軽く弾み、星のきらめきや「ナイスプレー！」などの応援メッセージと連動します。

演出中は次のブロック操作を一時的に止めるため、スマホ・iPadでも盤面が崩れにくいようにしています。

## 起動方法

このフォルダの `index.html` をブラウザで開きます。

ローカルサーバーを使う場合:

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開きます。

## 操作方法

| 操作 | キーボード |
| --- | --- |
| 左右移動 | ← / → |
| ソフトドロップ | ↓ |
| 回転 | ↑ / X |
| ハードドロップ | Space |
| ホールド | C |
| 一時停止 | P |
| リスタート | R |

スマホ・iPadでは画面下の操作ボタンで遊べます。キッズモードでは「左」「右」「回す」「下へ」「一気に落とす」「とっておく」のように、子ども向けのラベルになります。

## スマホ・iPadでの確認方法

1. ローカルサーバーを起動します。
2. PCと同じWi-FiにスマホやiPadを接続します。
3. PCのローカルIPアドレスを使って `http://PCのIPアドレス:8000/` を開きます。
4. 縦画面・横画面で盤面と操作ボタンが収まるか確認します。

## GitHub Pagesで公開する

1. このフォルダをGitHubリポジトリへpushします。
2. GitHubのリポジトリ設定で `Pages` を開きます。
3. `Deploy from a branch` を選び、公開したいブランチのルートを指定します。
4. 発行されたURLへアクセスします。

## ファイル構成

- `index.html`: モード選択とゲーム画面の構造
- `style.css`: ノーマル/キッズの見た目、レスポンシブUI、タッチ操作向けスタイル
- `script.js`: ゲームロジック、モード設定、描画、入力処理、効果音、ベストスコア保存
